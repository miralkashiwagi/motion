## 実装方法

### HTML

```html
<a href="#" class="btn">
  <span class="btn-text">ホバー</span>
  <span class="btn-icon btn-icon-east"></span>
</a>
```

### CSS

```css
.btn-icon{
  overflow: hidden;
  display: block;
  width: 1em;
  height: 1em;
}

.btn-icon-north-east::after{
  /* 略 */
}

.btn:hover .btn-icon-east::after {
  animation: move-icon-east 0.3s ease-in-out forwards;
}

@keyframes move-icon-east {
  0% {
    transform: translate(0,0);
  }
  50% {
    transform: translate(100%,0)
  }
  51% {
    transform: translate(-100%,0)
  }
  100% {
    transform: translate(0,0)
  }
}


```

## ポイント
- アイコン表示領域を、`overflow: hidden;` でクロップする
- `transform`プロパティを使用した `@keyframes` で使用して、アイコンを移動させる

## バリエーション
`@keyframes` の設定を変えると、アイコンの移動方向を変更できます。

### CSS

```css
//右上に向かって移動
@keyframes move-icon-north-east {
  0% {
    transform: translate(0,0);
  }
  50% {
    transform: translate(100%,-100%)
  }
  51% {
    transform: translate(-100%,100%)
  }
  100% {
    transform: translate(0,0)
  }
}

//右に向かって移動
@keyframes move-icon-east {
  0% {
    transform: translate(0,0);
  }
  50% {
    transform: translate(100%,0)
  }
  51% {
    transform: translate(-100%,0)
  }
  100% {
    transform: translate(0,0)
  }
}

//下に向かって移動
@keyframes move-icon-south {
  0% {
    transform: translate(0,0);
  }
  50% {
    transform: translate(0,100%)
  }
  51% {
    transform: translate(0,-100%)
  }
  100% {
    transform: translate(0,0)
  }
}

//左に向かって移動
@keyframes move-icon-west {
  0% {
    transform: translate(0,0);
  }
  50% {
    transform: translate(-100%,0)
  }
  51% {
    transform: translate(100%,0)
  }
  100% {
    transform: translate(0,0)
  }
}
```
