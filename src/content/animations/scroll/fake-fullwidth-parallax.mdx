## HTML構造

```html
<div class="fullwidth">
  <img src="img-sample-01.jpg" alt="">
  <div class="fullwidth-content">
      コンテンツ
  </div>
</div>
```

## CSS実装

```css
.fullwidth {
  width: 100%;
  height: 50vh;/* 任意の高さ */
  clip-path: polygon(0 0, 100% 0, 100% 100%, 0 100%);
}
.fullwidth img{
  /* 画像は画面いっぱいのサイズで配置 */
  position: fixed;
  inset: 0;
  width: 100%;
  height: 100%;
  object-fit: cover;
}

```

## 解説
- 画像は画面いっぱいのサイズで固定（position:fixed）しておきます。
- clip-pathで `.fullwidth` の要素の範囲でだけ画像を表示させています。



## バリエーション
- 画像だけでなく、コンテンツ部分まで固定（position:fixed）しても動作します。
